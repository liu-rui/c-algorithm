### 例 3.2 简单计算器
时间限制:1 秒
内存限制:32 兆
特殊判题:否
### 题目描述:
读入一个只包含 +, -, *, / 的非负整数计算表达式,计算该表达式的值。
### 输入:
测试输入包含若干测试用例,每个测试用例占一行,每行不超过 200 个字符,
整数和运算符之间用一个空格分隔。没有非法表达式。当一行中只有 0 时输入结束,相应的结果不要输出。
### 输出:
对每个测试用例输出 1 行,即该表达式的值,精确到小数点后 2 位。
### 样例输入:
```
1 + 2
4 + 2 * 5 - 7 / 11
0
```
### 样例输出:
```
3.00
13.36
```
